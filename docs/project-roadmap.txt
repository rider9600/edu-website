# 5. Purpose of Important Folders

## resources/

Contains:

* lecture PDFs
* markdown notes
* AI reference material
* concept planning material

Used for:

* AI agents
* article drafting
* visualization ideas
* concept extraction

Examples:
resources/lectures/L1.pdf
resources/markdown-notes/variance.md

IMPORTANT:
This folder is INTERNAL ONLY.

---

## content/

Contains final website articles.

Examples:
content/probability/conditional-probability.mdx

Each article should contain:

* intuition
* formulas
* examples
* visual explanation
* references

---

## visualizations/

# 7. Article Writing Structure

Every article should follow this structure:

1. Intuitive Introduction
2. Real-world Example
3. Formal Definition
4. Visual Explanation
5. Formula Intuition
6. Worked Example
7. Simulation / Graph
8. Common Mistakes
9. Related Topics
10. References
