# What is Probability

Probability is a measure of how likely an event is to occur. In simple terms, it is the fraction of times we would expect an event to happen if we repeated an experiment many times.

- Probability ranges from 0 (impossible) to 1 (certain).
- Often expressed as a percentage (0% to 100%).

```math
P(A) = \frac{\text{number of favorable outcomes}}{\text{total number of outcomes}}
```
